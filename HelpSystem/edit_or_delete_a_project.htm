Edit or Delete a Project

  


Editing a Project

To edit the details of a project from the Main Projects page, click on the name of the project you wish to edit. The Project Details page will open. Click on the pencil icon in the top right corner of the screen. A page will open that will allow you to edit all the various details of the projects, as well as to add notes and change the priority.

Deleting a Project

To delete the project, click 'Delete' in the bottom left of the Edit Project page. You will be asked if you are sure you want to delete the project. If you are, click OK.
