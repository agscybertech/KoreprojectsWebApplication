System Requirements

  


Kore Projects is not a program that you need to download and therefore there are no specific system requirements. To use Kore Projects, simply log on to the internet and enter the Kore Projects website address. The system will run using your web browser.

Depending on what version of your web browser you have, you may need to update it in order to use Kore Projects. If your computer asks you if you would like to update, say yes. Browsers that are compatible with Kore Projects include:

  • Internet Explorer (IE) 8 or newer
  • Mozilla Firefox (FF) 8 or newer
  • Safari 5 or newer
  • Google Chrome
  • iOS or Android - your default browser
  • All other devices - Opera Mini.
