System Setup

  


This allows you to customize the ‘background’ of your Kore Projects System, enabling you to create Project Worksheets quickly and easily.

 

  • Area – refers to where a job is taking place – eg: front garden
  • Item – refers to what is being worked on – eg: fountain replaced
  • Project Status – allows you to enter the different stages that a project might be at - eg: signing off. There are three default settings that cannot be altered – Scope, New Project and Complete. You are able to add as many more as you wish to your Project Status list. This is displayed on your projects page in order that everyone involved can see how a project is progressing.


To add an item in any of these three categories, click the blue ‘Add’ icon next to the appropriate heading. Enter the name and the display order. The display order is particularly useful when there is a set order to which you generally carry out your work.

Click ‘Add’ and the new area, item or project status will appear under the heading you chose.
